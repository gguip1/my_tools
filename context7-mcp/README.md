# Claude Context7 MCP

Claude Code에서 Context7을 사용하기 위한 MCP 설정

## 설정

```bash
cp .mcp.json.example .mcp.json
```

`.mcp.json` 열어서 `your-api-key` 부분을 실제 키로 교체

## Upstash API 키 발급

1. https://console.upstash.com 접속
2. 계정 생성 또는 로그인
3. Context7 서비스에서 API 키 발급
4. 발급된 키를 `.mcp.json`에 입력

## 기능

- 라이브러리 문서 실시간 검색
- 최신 API 문서 참조
- 코드 예제 제공
