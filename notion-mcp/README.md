# Claude Notion MCP

Claude Code에서 Notion API를 사용하기 위한 MCP 설정

## 설정

```bash
cp .mcp.json.example .mcp.json
```

`.mcp.json` 열어서 `YOUR_NOTION_API_KEY` 부분을 실제 키로 교체

## Notion API 키 발급

1. https://www.notion.so/my-integrations 접속
2. "새 API 통합" 생성
3. 발급된 키 복사 (ntn_xxxxx 형식)
4. 사용할 Notion 페이지에서 연결 추가 (페이지 ... > 연결 추가)
